## ADDED Requirements

### Requirement: User-Facing Biological Model Documentation

Colony SHALL explain the biological coordination model in contributor-facing documentation without making the documentation the behavior source of truth.

#### Scenario: contributor reads the model summary

- **WHEN** a contributor reads the README or Queen workflow documentation
- **THEN** the docs describe Queen publishing, worker pull, pheromone reinforcement and evaporation, local traces, and progressive disclosure in practical terms
- **AND** the docs link to the durable biological coordination OpenSpec contract for behavior details
